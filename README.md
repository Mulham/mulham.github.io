# قالب Chirpy

## المزايا

* سمتين داكنة وفاتحة
* لغة واجهة المستخدم المترجمة
* المنشورات المثبتة في الصفحة الرئيسية
* التصنيفات الهرمية
* الوسوم الشائعة
* جدول المحتويات
* تاريخ آخر تعديل
* تسليط الضوء على الكتابة وتنسيقاتها
* التعبيرات الرياضية
* مخططات حورية البحر والمخططات الانسيابية
* صور الوضع المظلم
* تضمين الوسائط
* أنظمة التعليقات
* بحث مدمج
* خلاصات feed xml
* PWA
* تحليلات الويب
* تحسين محركات البحث والأداء

# الاستخدام

قم بتنزيل ملفات الحزمة لجهازك ثم ضمن المجلد:

        bundle install

ثم

        bundle exec jekyll serve
